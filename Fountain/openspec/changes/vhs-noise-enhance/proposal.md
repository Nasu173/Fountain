# Proposal: VHS 噪点效果增强

## 背景

项目中已有 `Assets/VHS/VHSRetroFeature.shader` 和 `VHSRetroFeature.cs`，实现了完整的 VHS 复古滤镜。
当前噪点相关参数的可调范围偏小，在恐怖游戏场景中无法产生足够强烈的视觉冲击感。

## 问题

| 参数 | 当前范围 | 问题 |
|------|---------|------|
| `_GrainAmount` | 0 – 0.5 | 最大值偏低，颗粒感不够强烈 |
| `_GrainSize` | 0.5 – 3 | 颗粒尺寸上限不足，无法模拟粗粒胶片 |
| `_NoiseIntensity` | 0 – 0.3 | 整体噪声强度上限太低 |
| `_TapeNoise` | 0 – 0.2 | 磁带噪声上限不足 |
| Shader 内噪点混合系数 | 固定 `* 0.02` | 硬编码系数限制了实际输出幅度 |

## 目标

- 扩大上述参数的 `Range` 上限，让设计师可以调出更强烈的噪点效果
- 将 Shader 中硬编码的混合系数改为由参数驱动，消除隐藏的幅度限制
- 保持向后兼容：默认值不变，现有场景无需重新调参

## 不在范围内

- 不修改噪点算法本身（`hash` / `noise` / `fbm`）
- 不新增噪点类型
- 不改动 URP Renderer Feature 架构（现有 `OnRenderImage` 方式保持不变）
